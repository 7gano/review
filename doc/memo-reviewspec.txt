= ReVIEW記法について整理するためのメモ

== 記法の種類

 * 段落（本文）
 * 見出し（章・節・項・段）※4レベルまで？
 * コラム（見出しの変形）
 * 各種リスト（箇条書き、番号つき箇条書き、用語リスト）
 * 各種タグ（ブロックつき・ブロックなし）
 * 各種インライン
 * 著者用タグ（プリプロセッサ）

== 記法の用法

 * ブロックの入れ子は不可
 * インラインの入れ子は不可
 * ブロック内でのインラインの使用は可
 * インラインのエスケープには@<raw>を使う
 * ブロックのエスケープは困難なので行頭ではないところに書く？

=== 記法の課題（非互換）

 * ブロック内での段落の使用は許すか？
    * 現状ではLATEXBuilderは可、HTMLBuilderは不可
    * quoteブロック、leadブロックでは使えるようにしたい
 * listブロックやemlistブロック、cmdブロック内でのインラインは許すか？
    * 現状ではLATEXBuilderでは不可、HTMLBuilderでは可
    * LaTeXはvarbatim環境を使っているので無理？？ ←alltt環境なら可能？

== 利用できるタグ記法

全フォーマットで原則サポートしている（べき）基本記法と、それ以外の拡張記法に分ける。
(基本はHTMLBuilderとLATEXBuilderの実装状況から判断)

=== 基本

 * lead(read)
 * list
 * emlist
 * cmd
 * table
 * quote
 * image

 * footnote
 * (comment) ※ HTMLはdivで出力、LaTeXは%でコメントアウトで解釈が違う？

 * bibpaper  ※LaTeX非対応
 * doorquote  ※LaTeX非対応
 * talk  ※LaTeX非対応
 * address  ※LaTeX非対応
 * blockquote  ※LaTeX非対応
 * bpo  ※LaTeX非対応
 * flushright  ※LaTeXでは「right」になってる？？
 * note  ※LaTeX非対応
 * box  ※LaTeX非対応

 * numberlessimage  ※LaTeX非対応
 * hr  ※LaTeX非対応
 * parasep  ※HTMLでは<br/>で対応
 * label  ※HTML非対応、EPUBBuilderで<a id="..." />で対応
